# Introduction

**Simple Accounting** is a web-based application designed to help individuals and businesses generate accurate accounting reports quickly and effortlessly.

Managing financial records can be complex and time-consuming. Simple Accounting simplifies this process by allowing users to import their transaction data directly from an Excel file. Once uploaded, the system automatically processes the data and generates essential accounting reports without requiring advanced accounting knowledge or manual journal entry.

**With Simple Accounting, users can:**

- [Import](/journals/import) transaction data from other apps in Excel / CSV Format
- Automatically generate an [Accounting Report](/accounting-reports/about): 
  - [General Ledger](/accounting-reports/general-ledger)
  - [Subledger](/accounting-reports/subledger)
  - [Trial Balance](/accounting-reports/trial-balance)
  - [Profit & Loss](/accounting-reports/profit-and-loss)
  - [Balance Sheet](/accounting-reports/balance-sheet)
- Use these reports for business analysis and decision-making
