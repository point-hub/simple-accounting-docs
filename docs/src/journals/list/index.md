# List Journal

The List Journal feature allows users to view and manage recorded journal entries within the system. It provides a structured and organized display of journal data, enabling users to review transactions efficiently.

The journal list supports pagination to ensure optimal performance and usability when handling large volumes of data. Users can also filter and sort journal entries to quickly locate specific transactions based on relevant criteria.

## Scenarios

- **Success Scenarios**
  1. Display paginated journal data.
  2. Filter journal data by date.
  3. Filter journal data by form_number.
  4. Sort journal data by date.
  5. Sort journal data by form_number.
- **Failure Scenarios**
  1. User is not authenticated.
